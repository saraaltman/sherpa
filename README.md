# sherpa

To run the frontend: 
- `cd sherpa`
- `yarn dev`
-  page loads at http://localhost:5173/


To run the backend:
- `cd api`
- `npm start`
- port is 3000

To connect to db:
- create .env file under the /api directory
- add the following line:
`MONGODB_URI=mongodb+srv://{USER}:{PASSWORD}@cluster0.69idt.azure.mongodb.net/?retryWrites=true&w=majority`
